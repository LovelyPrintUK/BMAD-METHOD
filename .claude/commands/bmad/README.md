# BMAD Method - Slash Commands

This directory contains slash commands for the BMAD Method (Build More, Architect Dreams).

## Available Commands

### Agents

- `/bmad:bmm:agents:analyst` - Business Analyst agent for Discovery workflows

### Workflows

- `/bmad:bmm:workflows:product-brief` - Product Brief Discovery workflow

## How to Use

1. Open a new chat in Claude Code
2. Type one of the slash commands above
3. Press Enter
4. Follow the guided workflow

## Output Location

Discovery documents are saved to: `docs/discovery/`
